## Возможности

- Управление активными сеансами (начало/завершение, авторасчёт стоимости)
- Управление бронированиями столов
- Гибкая настройка тарифов (по дням и часам)
- Отчёты по выручке и загруженности
- Защита от брутфорса (блокировка после 3 ошибок)

## Установка

### Требования
- Windows 10/11
- .NET 8 Runtime → https://dotnet.microsoft.com/download/dotnet/8.0
- Node.js 18+ → https://nodejs.org

### Шаги

# 1. Открыть папку с проектом

# 2. Запустить сборку
build.bat

# 3. Готовый файл в папке publish\

## Запуск

# Двойной клик
publish\AnticafeBackend.exe

Открыть браузер: http://localhost:5154

### Вход

Логин: admin
Пароль: admin

> ⚠️ После первого входа смените пароль!


## Устранение проблем

| Проблема | Решение |
|----------|---------|
| Порт занят | `netstat -ano | findstr :5154` → убить процесс |
| БД не создаётся | Запустить от администратора |
| Не открывается браузер | Вручную http://localhost:5154 |
